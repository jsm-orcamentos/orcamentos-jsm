INSTRUÇÕES PARA O SISTEMA DE ORÇAMENTOS JSM

1. COPIE TODOS OS ARQUIVOS PARA UMA PASTA LOCAL
2. ABRA index.html NO NAVEGADOR
3. SIGA AS ETAPAS PARA CRIAR UM ORÇAMENTO
4. PARA CARREGAR UM ORÇAMENTO EXISTENTE, CLIQUE EM "CARREGAR ORÇAMENTO EXISTENTE"
5. DIGITE O NOME DO CLIENTE PARA BUSCAR
6. SELECIONE O ORÇAMENTO DESEJADO
7. EDITE SE NECESSÁRIO E GERE O PDF

PARA INTEGRAR COM GOOGLE DRIVE:
1. CRIE UM PROJETO NO GOOGLE CLOUD CONSOLE
2. ATIVE A GOOGLE DRIVE API
3. CRIE CREDENCIAIS OAUTH 2.0
4. SUBSTITUA OS PLACEHOLDERS NO script.js COM SUAS CREDENCIAIS
5. IMPLEMENTE AS FUNÇÕES salvarNoDrive E carregarDoDrive

CORES UTILIZADAS:
- Azul escuro: #0A1A3D
- Laranja: #F47A20
- Branco: #FFFFFF

FORMATO DE DATAS:
- Exibição: DD/MM/AAAA
- Arquivos: DD-MM-AAAA

NOMES DE ARQUIVOS:
- JSON: NomeCliente — DD-MM-AAAA.json
- PDF: JSM_Orçamento_NomeCliente_DD-MM-AAAA.pdf
